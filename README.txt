APP - New Roots 

New Roots is an application that will utilize the zillow api to help people find homes in an area that they are moving too.  

Frameworks 
	- Code Igniter
	- JQuery
	- Bootstrap

Database is located in the Assets folder as well as concepts for the site look.

You can download the files by clicking on the Zip tab and place the files on your local server. 

default path ie http://YourLocalhost/MDD/index.php
